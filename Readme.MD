# Pokémon Starters Battle
A 2D retro-style Pokémon-inspired game made in C++ using SFML. Choose your Pokémon, explore the world, and battle the first 3 starter pokemons ( and pikachu)

## 🎮 How to Play

1. Use arrow keys to navigate the overworld.
2. Walk into zones to encounter Pokémon.
3. Choose to fight or flee.
4. Defeat Pokémon and return to explore more!

## 🛠️ Build Instructions

### Requirements

- SFML (Simple and Fast Multimedia Library)
- A C++17 compatible compiler

### Linux/macOS:

```bash
sudo apt install libsfml-dev
make overworld
./overworld

### Play via latest release package
